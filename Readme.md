
# Gentilic Names Suffixation In Hebrew

###### *This test was made as part of The Phonology-Morphology Interface seminar, Tel-Aviv University 2021*

To run the experiment clone this repo, then open *index.html* in a web browser (double click).
<br>Click the "next" button to start and "play" to play a sound.
<br>To change version (2 versions are available) change **exp** in *js/main.js* to 1 or 2. 

The *instructions.html* file contains the instructions that were presented and read to the participants at the beginning:
> בניסוי הבא תשמעו 30 מילים מומצאות המבטאות שיוך של אדם למדינה כלשהי.
לדוגמה, המילה (האמתית) "איטלקי" משמעותה איש מאיטליה, המילה (המומצאת) שושי משמעותה איש משוש.
במהלך הניסוי תצטרכו להטות את המילים לפי ההוראות שיופיעו על המסך
אנא אמרו היא/הם/הן לפי ההוראה ולאחריה את המילה כפי שלדעתכם יש להטות אותה
למשל, המילה הנשמעת היא "איטלקי" וההוראה: "הטי/הטה לצורת הרבים"
הם...
המילה הנשמעת היא שושי וההוראה: "הטי/הטה לצורת הנקבה"
היא...
המילים מופיעות בסדר אקראי, כל התשובות נכונות, אנא ענו את התשובה הראשונה שאתם חושבים עליה.
